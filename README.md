# Flamingo Florist Website

Beautiful custom floral arrangements for weddings, events, and special occasions - 30 years of making moments special.

## About

Flamingo Florist has been serving the community for over 30 years, creating stunning floral arrangements that make people smile, cry happy tears, and feel truly special. We don't create cookie-cutter arrangements - we craft works of art that tell a story.

## Features

- **Custom Wedding Arrangements**: Bridal bouquets, ceremony decorations, and reception centerpieces
- **Special Event Flowers**: Corporate events, birthday celebrations, and custom arrangements
- **Memorial Services**: Compassionate floral tributes for funeral and memorial services
- **Gift Baskets**: Curated seasonal arrangements and custom gift baskets
- **Local Delivery**: Serving our local community with fresh, beautiful flowers

## Getting Started

### Prerequisites

- Node.js 18 or higher
- pnpm package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd flamingo-florist
```

2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Building for Production

```bash
pnpm build
pnpm start
```
